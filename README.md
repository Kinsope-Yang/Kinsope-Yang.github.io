## Krei 编程语言

Krei 编程语言，简称K语言，是Kinsope于2021年7月23日开始开发的语言。

### 代码格式

由于此语言刚刚开始开发，所以只是制定好了初步的语法：

```markdown
# main.ke（此行为注释）

fun main(){
  var a;#弱类型变量
  a = 100;#此时a为int类型
  output("a = ",a);
  input(a);
  output("a = ",a);
}
```
